B — Tableau global + détail (Recommandé)

Toutes les réservations dans un tableau filtrable. Clic sur une ligne → panneau de détail : dossier complet, consentements, actions (re-envoyer email, marquer présente, annuler). Plus puissant pour gérer beaucoup d'élèves.
